# PokemonMoveDatabase

此專案旨在建立一個包含寶可夢及其招式的資料庫系統。資料庫包含三個資料表，分別記錄寶可夢的招式、招式的屬性與威力，以及屬性之間的克制關係。使用 WinForm 介面顯示資料表內容並進行基本的資料庫操作（插入、更新、刪除及查詢）。

## 資料庫結構
- **寶可夢_招式**：記錄寶可夢及其對應的招式（Primary Key：寶可夢）。
- **招式_屬性_威力**：記錄每個招式的屬性及威力（Primary Key：招式）。
- **屬性_克制屬性_被克制屬性**：記錄屬性之間的相剋關係（Primary Key：屬性）。

## 功能
1. **資料操作**
   - **插入資料（INSERT）**：新增寶可夢及其招式、招式屬性與威力、屬性克制關係的資料。
   - **更新資料（UPDATE）**：更新現有資料，如更改寶可夢的招式或調整招式的威力。
   - **刪除資料（DELETE）**：刪除特定寶可夢的資料。
   - **查詢資料（SELECT）**：查詢並顯示資料表中的內容。

2. **資料關聯設定**
   - 建立寶可夢、招式與屬性之間的關聯。
   - 在 WinForm 介面中，點擊「寶可夢_招式」資料表中的項目時，會顯示對應的招式屬性與屬性克制關係。

3. **使用介面**
   - 在 WinForm 上建立三個 DataGridView 顯示資料表內容，並動態更新顯示的資料。

## 使用說明
1. **插入與查詢資料**
   - 在資料庫中插入預設資料：
     - 寶可夢_招式資料表：新葉喵 - 樹葉、呆火鱷 - 火花、潤水鴨 - 水槍
     - 招式_屬性_威力資料表：樹葉 - 草 - 40、火花 - 火 - 40、水槍 - 水 - 40
     - 屬性_克制屬性_被克制屬性資料表：草 - 水 - 火、火 - 草 - 水、水 - 火 - 草
   - 使用 SELECT 指令查詢所有資料。

2. **更新與查詢資料**
   - 更新寶可夢的招式及其威力：
     - 將潤水鴨的招式改為「水流噴射」。
     - 將火花的威力改為「60」。
   - 查詢指定資料列，顯示特定欄位的資料。

3. **刪除與查詢資料**
   - 刪除潤水鴨的資料並查詢指定條件的資料。

4. **關聯與顯示**
   - 在 WinForm 上建立關聯，當點選寶可夢資料時，對應的招式屬性和屬性克制資料會同步顯示。
